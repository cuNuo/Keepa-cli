### 3. Graphimage binary transport

- 做了什么：为 `KeepaClient.request` 增加 `binary` 与 `out` 参数，新增 `_live_binary_response` 写 PNG 文件；CLI 增加 `graphs image --out`。
- 为什么这么做：`/graphimage` 返回 PNG bytes，不能直接进入 JSON stdout；必须显式文件输出。
- 结果：无 `--out` 的 live graphimage 返回 `binary_output_path_required`；提供 `--out` 时可写入文件并返回 `bytes_written`、`content_type` 与 `cache_provenance`。
