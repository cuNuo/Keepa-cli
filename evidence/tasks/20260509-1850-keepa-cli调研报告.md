## 关键发现

- Keepa API 基础地址为 `https://api.keepa.com/`，请求带 `key` 参数。
- 响应会返回 `refillRate`、`refillIn`、`tokensLeft`、`tokensConsumed`、`tokenFlowReduction` 和 `error`。

- Product Request 常规成本为 1 token / ASIN；`offers`、`finder`、`bestsellers`、`topseller` 等需要显式预算保护。
- 当前机器有 Python、uv、Node、gh、git；未发现 Rust 工具链。
