# Keepa-cli evidence 索引

## 目录约定

- `tasks/`：任务日志，记录目标、输入、过程、验证、风险和结论。
- `manifest.csv`：任务日志清单，便于 Agent 快速检索最近证据。

## 命名约定

- 任务日志使用 `YYYYMMDD-HHMM-主题.md`。
- 主体默认使用中文；命令、路径、API 字段和错误码保留原文。
- 单次任务只保留一份主日志，后续更新直接追加到同一文件。

## 写入要求

- 每份日志至少包含：任务目标、背景与输入、处理过程、验证结果、关联产物、风险与后续、结论。
- 验证结果必须写明具体命令和结论，不只写“已验证”。
- 若任务产出长期稳定结论，应在日志中标注对应 Serena memory。
- 新增或更新任务日志后，应同步维护 `manifest.csv`。
